# express-template
Express.js&Typescriptとnode-jdbcによるDBアクセスを伴うRESTサーバーのテンプレートです

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Lints and fixes files
```
npm run lint
```

### Compiles for production
```
npm run build
```

### Run your unit tests
```
npm test
```

### Run your REST server for production
```
npm start
```
